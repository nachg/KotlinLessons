# Классификация элементов языка Kotlin

| Категория               | Элемент                                                                                             | Краткое описание                                                                                                                            |
|-------------------------|-----------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| **Ключевые слова**      | `package`, `import`                                                                                 | Объявление пакета и импорт зависимостей.                                                                                                    |
| **Ключевые слова**      | `class`, `interface`, `object`                                                                       | Определение класса, интерфейса, синглтона (object).                                                                                         |
| **Ключевые слова**      | `fun`                                                                                               | Объявление функции.                                                                                                                         |
| **Ключевые слова**      | `val`, `var`                                                                                        | Неизменяемая (val) и изменяемая (var) переменные.                                                                                           |
| **Ключевые слова**      | `if`, `else`                                                                                        | Условный оператор.                                                                                                                          |
| **Ключевые слова**      | `when`                                                                                              | Замена switch с мощными возможностями сопоставления.                                                                                        |
| **Ключевые слова**      | `for`, `while`, `do`                                                                                | Циклы.                                                                                                                                      |
| **Ключевые слова**      | `break`, `continue`, `return`                                                                       | Операторы прерывания цикла и выхода из функции.                                                                                             |
| **Ключевые слова**      | `try`, `catch`, `finally`, `throw`                                                                  | Обработка исключений.                                                                                                                       |
| **Ключевые слова**      | `typealias`, `inline`, `noinline`                                                                   | Псевдоним типа, модификаторы встраивания.                                                                                                   |
| **Ключевые слова**      | `data`, `sealed`, `enum`, `annotation`                                                              | Специализированные классы: data-классы, sealed-классы, перечисления, аннотации.                                                            |
| **Ключевые слова**      | `open`, `abstract`, `override`, `final`                                                             | Модификаторы наследования и переопределения.                                                                                                |
| **Ключевые слова**      | `internal`, `private`, `protected`, `public`                                                        | Модификаторы видимости.                                                                                                                     |
| **Ключевые слова**      | `companion`                                                                                         | Объявление сопутствующего объекта (аналог статических членов).                                                                              |
| **Ключевые слова**      | `init`                                                                                              | Блок инициализации класса.                                                                                                                  |
| **Ключевые слова**      | `this`, `super`                                                                                     | Ссылки на текущий объект и родительский класс.                                                                                              |
| **Ключевые слова**      | `is`, `!is`, `as`, `as?`                                                                            | Проверка и приведение типов.                                                                                                                |
| **Ключевые слова**      | `in`, `!in`                                                                                         | Проверка принадлежности диапазону/коллекции.                                                                                                |
| **Ключевые слова**      | `out`, `in` (вариативность)                                                                         | Проекция типов (ковариантность и контравариантность).                                                                                       |
| **Ключевые слова**      | `reified`                                                                                           | Вещественный тип параметра (в inline-функциях).                                                                                             |
| **Ключевые слова**      | `suspend`                                                                                           | Маркер приостанавливаемой функции (корутины).                                                                                               |
| **Ключевые слова**      | `tailrec`                                                                                           | Оптимизация хвостовой рекурсии.                                                                                                             |
| **Ключевые слова**      | `operator`                                                                                          | Перегрузка операторов.                                                                                                                      |
| **Ключевые слова**      | `infix`                                                                                             | Вызов функции в инфиксной форме.                                                                                                            |
| **Операторы**           | `+`, `-`, `*`, `/`, `%`                                                                             | Арифметические операции.                                                                                                                    |
| **Операторы**           | `+=`, `-=`, `*=`, `/=`, `%=`                                                                        | Присваивание с операцией.                                                                                                                   |
| **Операторы**           | `++`, `--`                                                                                          | Инкремент/декремент.                                                                                                                        |
| **Операторы**           | `==`, `!=`                                                                                          | Сравнение по значению (структурное).                                                                                                        |
| **Операторы**           | `===`, `!==`                                                                                        | Сравнение по ссылке.                                                                                                                        |
| **Операторы**           | `<`, `>`, `<=`, `>=`                                                                                | Сравнение чисел.                                                                                                                            |
| **Операторы**           | `&&`, `\|\|`, `!`                                                                                   | Логические И, ИЛИ, НЕ.                                                                                                                      |
| **Операторы**           | `?.`                                                                                                | Безопасный вызов (если объект не null, выполнить).                                                                                          |
| **Операторы**           | `?:`                                                                                                | Элвис-оператор (значение по умолчанию при null).                                                                                            |
| **Операторы**           | `!!`                                                                                                | Принудительный вызов (бросает NPE если null).                                                                                               |
| **Операторы**           | `..`                                                                                                | Оператор создания диапазона.                                                                                                                |
| **Операторы**           | `in`, `!in`                                                                                         | Проверка вхождения в диапазон/коллекцию.                                                                                                    |
| **Операторы**           | `[]` (get/set)                                                                                      | Доступ по индексу (перегружается).                                                                                                          |
| **Операторы**           | `::`                                                                                                | Ссылка на член (функцию, свойство, класс).                                                                                                  |
| **Операторы**           | `@`                                                                                                 | Обозначение метки (например, для break или return).                                                                                         |
| **Операторы**           | `->`                                                                                                | Разделитель в when-выражениях и лямбда-выражениях.                                                                                          |
| **Типы данных**         | `Byte`, `Short`, `Int`, `Long`                                                                      | Целочисленные типы (8,16,32,64 бит).                                                                                                        |
| **Типы данных**         | `Float`, `Double`                                                                                   | Числа с плавающей точкой (32 и 64 бит).                                                                                                     |
| **Типы данных**         | `Boolean`                                                                                           | Логический тип (true/false).                                                                                                                |
| **Типы данных**         | `Char`                                                                                              | Символ (16 бит).                                                                                                                            |
| **Типы данных**         | `String`                                                                                            | Строка.                                                                                                                                     |
| **Типы данных**         | `Any`                                                                                               | Корневой тип (аналог Object в Java).                                                                                                        |
| **Типы данных**         | `Unit`                                                                                              | Тип, означающий отсутствие полезного значения (аналог void).                                                                                |
| **Типы данных**         | `Nothing`                                                                                           | Тип, который никогда не может иметь значения (используется в функциях, всегда бросающих исключение).                                       |
| **Коллекции**           | `List`, `MutableList`                                                                               | Упорядоченная коллекция (интерфейсы).                                                                                                       |
| **Коллекции**           | `Set`, `MutableSet`                                                                                 | Множество уникальных элементов (интерфейсы).                                                                                                |
| **Коллекции**           | `Map`, `MutableMap`                                                                                 | Словарь ключ-значение (интерфейсы).                                                                                                         |
| **Коллекции**           | `ArrayList`, `LinkedList`                                                                           | Реализации списка.                                                                                                                          |
| **Коллекции**           | `HashSet`, `LinkedHashSet`, `TreeSet`                                                               | Реализации множества.                                                                                                                       |
| **Коллекции**           | `HashMap`, `LinkedHashMap`, `TreeMap`                                                               | Реализации словаря.                                                                                                                         |
| **Коллекции**           | `Sequence`                                                                                          | Ленивые коллекции для обработки больших данных.                                                                                             |
| **Функции станд. библиотеки** | `let`, `apply`, `run`, `with`, `also`                                                         | Функции области видимости (scope functions) для работы с объектами.                                                                         |
| **Функции станд. библиотеки** | `repeat`                                                                                        | Повторение действия заданное число раз.                                                                                                     |
| **Функции станд. библиотеки** | `TODO`                                                                                          | Маркер нереализованного кода (бросает исключение).                                                                                          |
| **Функции станд. библиотеки** | `lazy`                                                                                          | Делегат для ленивой инициализации.                                                                                                          |
| **Функции станд. библиотеки** | `use`                                                                                           | Автоматическое закрытие ресурсов (Closeable).                                                                                               |
| **Функции станд. библиотеки** | `require`, `check`, `error`                                                                    | Проверка условий и генерация исключений.                                                                                                    |
| **Функции станд. библиотеки** | `filter`, `map`, `flatMap`, `fold`, `reduce`                                                   | Основные операции над коллекциями.                                                                                                          |
| **Функции станд. библиотеки** | `forEach`, `forEachIndexed`                                                                    | Итерация по элементам.                                                                                                                      |
| **Функции станд. библиотеки** | `groupBy`, `associate`                                                                         | Преобразование коллекций в словари.                                                                                                         |
| **Функции станд. библиотеки** | `take`, `drop`, `chunked`                                                                      | Выделение части коллекции.                                                                                                                  |
| **Функции станд. библиотеки** | `any`, `all`, `none`                                                                           | Проверка условий на коллекции.                                                                                                              |
| **Функции станд. библиотеки** | `zip`, `unzip`                                                                                 | Объединение/разъединение коллекций.                                                                                                         |
| **Функции станд. библиотеки** | `sorted`, `sortedBy`                                                                           | Сортировка коллекций.                                                                                                                       |
| **Функции станд. библиотеки** | `toList`, `toSet`, `toMap`                                                                     | Преобразование в другой тип коллекции.                                                                                                      |
| **Функции станд. библиотеки** | `withIndex`                                                                                    | Получение индекса вместе со значением.                                                                                                      |

Эта таблица охватывает наиболее употребимые элементы языка Kotlin. Для углублённого изучения рекомендуется обратиться к официальной документации.
