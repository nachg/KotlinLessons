# Классификация элементов языка Kotlin

| Категория               | Элемент                                                                                             | Краткое описание                                                                                                                            |
|-------------------------|-----------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| **Ключевые слова**      | `package`, `import`                                                                                 | Объявление пакета и импорт зависимостей.                                                                                                    |
| **Ключевые слова**      | `fun`                                                                                               | Объявление функции.                                                                                                                         |
| **Ключевые слова**      | `val`, `var`                                                                                        | Неизменяемая (val) и изменяемая (var) переменные.                                                                                           |
| **Ключевые слова**      | `if`, `else`                                                                                        | Условный оператор.                                                                                                                          |
| **Ключевые слова**      | `when`                                                                                              | Замена switch с мощными возможностями сопоставления.                                                                                        |
| **Ключевые слова**      | `for`, `while`, `do`                                                                                | Циклы.                                                                                                                                      |
| **Ключевые слова**      | `break`, `continue`, `return`                                                                       | Операторы прерывания цикла и выхода из функции.                                                                                             |
| **Ключевые слова**      | `in`, `!in`                                                                                         | Проверка принадлежности диапазону/коллекции.                                                                                                |
| **Операторы**           | `+`, `-`, `*`, `/`, `%`                                                                             | Арифметические операции.                                                                                                                    |
| **Операторы**           | `+=`, `-=`, `*=`, `/=`, `%=`                                                                        | Присваивание с операцией.                                                                                                                   |
| **Операторы**           | `++`, `--`                                                                                          | Инкремент/декремент.                                                                                                                        |
| **Операторы**           | `==`, `!=`                                                                                          | Сравнение по значению (структурное).                                                                                                        |
| **Операторы**           | `===`, `!==`                                                                                        | Сравнение по ссылке.                                                                                                                        |
| **Операторы**           | `<`, `>`, `<=`, `>=`                                                                                | Сравнение чисел.                                                                                                                            |
| **Операторы**           | `&&`, `\|\|`, `!`                                                                                   | Логические И, ИЛИ, НЕ.                                                                                                                      |
| **Операторы**           | `?.`                                                                                                | Безопасный вызов (если объект не null, выполнить).                                                                                          |
| **Операторы**           | `?:`                                                                                                | Элвис-оператор (значение по умолчанию при null).                                                                                            |
| **Операторы**           | `!!`                                                                                                | Принудительный вызов (бросает NPE если null).                                                                                               |
| **Операторы**           | `..`                                                                                                | Оператор создания диапазона.                                                                                                                |
| **Операторы**           | `in`, `!in`                                                                                         | Проверка вхождения в диапазон/коллекцию.                                                                                                    |
| **Операторы**           | `[]` (get/set)                                                                                      | Доступ по индексу (перегружается).                                                                                                          |
| **Операторы**           | `::`                                                                                                | Ссылка на член (функцию, свойство, класс).                                                                                                  |
| **Операторы**           | `@`                                                                                                 | Обозначение метки (например, для break или return).                                                                                         |
| **Операторы**           | `->`                                                                                                | Разделитель в when-выражениях и лямбда-выражениях.                                                                                          |
| **Типы данных**         | `Byte`, `Short`, `Int`, `Long`                                                                      | Целочисленные типы (8,16,32,64 бит).                                                                                                        |
| **Типы данных**         | `Float`, `Double`                                                                                   | Числа с плавающей точкой (32 и 64 бит).                                                                                                     |
| **Типы данных**         | `Boolean`                                                                                           | Логический тип (true/false).                                                                                                                |
| **Типы данных**         | `Char`                                                                                              | Символ (16 бит).                                                                                                                            |
| **Типы данных**         | `String`                                                                                            | Строка.                                                                                                                                     |
| **Типы данных**         | `Any`                                                                                               | Корневой тип (аналог Object в Java).                                                                                                        |
| **Типы данных**         | `Unit`                                                                                              | Тип, означающий отсутствие полезного значения (аналог void).                                                                                |
| **Коллекции**           | `List`, `MutableList`                                                                               | Упорядоченная коллекция (интерфейсы).                                                                                                       |
